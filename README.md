## descreption

هذا المشروع عبارة عن صفحة ويب أحادية (Single-Page Website) تم تطويرها باستخدام **HTML5** و **CSS3** فقط، بناءً على تصميم مطعم "دلوعة" (Dalouna).

الهدف الأساسي من المشروع هو تطبيق مفاهيم الويب الأساسية، مع التركيز على:

* **HTML  (Semantic HTML):** استخدام عناصر مثل `<header>`, `<main>`, `<section>`, و `<footer>`.
* * (Flexbox Layout) : بناء التصميم الشبكي (Grid) وتنسيق الأقسام باستخدام Flexbox لضمان سهولة الترتيب.
* **(Responsive Design):** ضمان عرض الصفحة بشكل مناسب على كل من أجهزة الكمبيوتر المكتبية (Desktop) والأجهزة المحمولة (Mobile).
*  Styling Tokens:** تطبيق الثوابت والمتغيرات (CSS Variables) لإدارة الألوان والمسافات.
